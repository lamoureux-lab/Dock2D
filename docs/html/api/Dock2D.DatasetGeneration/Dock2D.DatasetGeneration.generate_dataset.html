generate_datasets(protein_pool, num_proteins=None)

Generate docking and interaction dataset based on protein pool.

Parameters

  • protein_pool – protein pool .pkl

  • num_proteins – can specify size of protein pool to use in generating pairwise interactions, None uses the entire protein pool.

Returns

fft_score_list used only in plotting, the docking_set is the docking dataset (IP) as a list of lists [receptor, ligand, rot, trans], and interaction_set a list of [protein_shapes, interactions_list, labels_list]

generate_interactions(receptor, ligand)

Generate pairwise interactions through FFT scoring of shape bulk and boundary.

Parameters

  • receptor – a shape assigned as receptor from protein pool

  • ligand – a shape assigned as ligand from protein pool

Returns

receptor, ligand, and their fft_score

generate_pool(params, pool_savefile, num_proteins=500)

Generate the protein pool using parameters for concavity and number of points.

Parameters

  • params –

    Pool generation parameters as a list of tuples for alpha and number of points as (value, relative freq).

    shape_alpha = [(0.70, 1), (0.80, 4), (0.90, 6), (0.95, 4), (0.98, 1)]
num_points = [(40, 1), (60, 3), (80, 3), (100, 1)]

  • pool_savefile – protein pool .pkl filename

  • num_proteins – number of unique protein shapes to make

Returns

stats as observed parameter list of tuples (value, probs)

plot_accepted_rejected_shapes(receptor, ligand, rot, trans, lowest_energy, fft_score, protein_pool_prefix, plot_count)

Plot examples of accepted and rejected shape pairs, based on docking and interaction decision thresholds set.

Parameters

  • receptor – receptor shape image

  • ligand – ligand shape image

  • rot – rotation to apply to ligand

  • trans – translation to apply to ligand

  • lowest_energy – minimum energy from FFT score

  • fft_score – over all FFT score

  • protein_pool_prefix – filename prefix

  • plot_count – used as index in plotting

plot_energy_distributions(train_fft_score_list, test_fft_score_list, show=False)

Plot histograms of all pairwise energies within training and testing set.

Parameters

  • train_fft_score_list – training set FFT scores

  • test_fft_score_list – test set FFT scores

  • trainpool_num_proteins – number of proteins in training pool

  • testpool_num_proteins – number of proteins in testing pool

  • show – show plot in new window (does not affect plot saving)

run_generator()

Generates the training, validation, and testing sets for both docking (IP) and interaction (FI) from. Write all datasets to .pkl files. Saves all metrics to file. Prints IP and FI dataset stats. If self.plotting=True plot and save dataset generation plots. Specify self.show=True to show each plot in a new window (does not affect saving).
